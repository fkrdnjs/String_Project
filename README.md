# String_Project
- string.h 헤더파일을 직접 구현

## 구현 함수
- my_strlen(): 문자열의 길이 반환
- my_strcpy(): 문자열을 복사
- my_strncpy(): 문자열 일부를 다른 문자열로 복사
- my_strcat(): 두 개의 문자열 결합
- my_strncat(): 문자열 일부를 다른 문자열과 결합
- my_strcmp(): 두 개의 문자열 비교
- my_strncmp(): 두 개의 문자열을 지정된 길이만큼 비교
- my_strchr(): 문자열에서 특정 문자 찾기
